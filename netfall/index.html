The net module

Every script talks to the grid through import net. Calls marked blocking genuinely pause your script until the work is done — that is the game. Each function you use adds to a script's RAM cost (base 1.0GB).

Script engine offline. This page is not cross-origin isolated, so the Python interpreter can't start. See Troubleshooting below.

Working a server (blocking)

net.hack(host)Siphon credits from a rooted server. Returns credits gained; a failed attempt returns 0 and raises the target's security slightly. Needs root and enough skill. +0.15GB
net.grow(host)Fluff a server's balance back up (security creeps up). Returns the new balance. +0.15GB
net.weaken(host)Scrub security down toward its floor. Returns the new security level. +0.15GB
net.sleep(seconds)Do nothing for a while. free

Reading the grid

net.scan(host)List of hostnames wired to host (discovers them). +0.25GB
net.server(host)Dict of facts: money, max_money, security, min_security, level_required, ports_required, rooted, ram, ram_used, hostname. +0.05GB
net.servers()Every hostname you've discovered. +0.3GB
net.money() / net.skill()Your balance / your skill level. +0.02GB each
net.has_root(host)True if you have root. +0.05GB
net.hack_time(host) etc.grow_time, weaken_time: seconds the op would take right now. +0.05GB each
net.hostname() / net.argsWhere this script runs / the arguments it was started with (a list of strings). free

Root & automation

net.nuke(host)Force root, if you own crackers for every lock. Raises net.NetError if you can't. +0.05GB
net.run(script, host, *args)Launch a script from your rig onto any rooted server (its RAM, your payroll). Returns the pid, or 0 if it couldn't start. +1.0GB
net.ps() / net.kill(pid)List running scripts / stop one. +0.3GB each
net.log(*parts)Write to this script's log — same place print() goes. free

How the numbers move

  • Security rises when you hack (+0.6) or grow (+0.3), and falls when you weaken (−1.5), never below the server's floor. High security means slower operations and worse hack odds.
  • Hacking steals a percentage of the server's current credits — keep the balance high with grow or the takes shrink.
  • Skill grows with every completed operation. Higher skill: better odds, bigger cuts, faster work, deeper servers.
  • RAM limits how much runs at once. Scripts run anywhere you have root; files stay on your rig.

Errors

Grid refusals raise net.NetError — catch them like any exception:

try:
    net.nuke("night-owl")
except net.NetError as e:
    print(f"not yet: {e}")

Troubleshooting

  • “Script engine offline / not cross-origin isolated” — the game needs SharedArrayBuffer, which needs COOP/COEP headers. A bundled service worker adds them and reloads once automatically; if it can't (private browsing with service workers blocked, very old browser), scripts can't run. A normal window in a current Chrome, Firefox, Edge, or Safari works.
  • Scripts stop when the tab closes — yes: your rig is your browser tab. Siphon drones keep earning while you're away (up to 4 hours).
  • Save data lives in this browser's localStorage. reset confirm wipes it.
